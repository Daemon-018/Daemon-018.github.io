# 🔑 ADD GITHUB SECRET — DO THIS STEP

## Add OpenRouter API Key as GitHub Secret

1. Go to: https://github.com/Daemon-018/Daemon-018.github.io/settings/secrets/actions

2. Click **"New repository secret"**

3. **Name:** `OPENROUTER_API_KEY`

4. **Value:** 
```
sk-or-...9aae
```

5. Click **"Add secret"**

---

## Enable GitHub Pages

1. Go to: https://github.com/Daemon-018/Daemon-018.github.io/settings/pages

2. **Source:** Deploy from a branch

3. **Branch:** master / (root)

4. Click **"Save"**

---

## Trigger First AI Post

1. Go to: https://github.com/Daemon-018/Daemon-018.github.io/actions

2. Click **"SecHunter AI Blog Engine"** workflow

3. Click **"Run workflow"** → **"Run workflow"**

4. Wait 2-3 minutes for the AI to generate a post

5. Check: https://Daemon-018.github.io

---

*Once done, the AI engine will run automatically every day at 6:00 AM IST.*
*No need to keep your phone on. GitHub runs everything.*
